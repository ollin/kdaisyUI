# Tutorials

Step-by-step lessons to learn kdaisyui from the ground up.

## Learning path

| Tutorial | What you'll learn | Time |
|----------|-------------------|------|
| [Getting started](getting-started.md) | Set up a project, render your first components, understand the DSL | 10 min |
| [Build a dashboard with Ktor](build-a-dashboard.md) | Create a full admin dashboard with htmx for progressive loading | 30 min |

## Recommended order

1. **Start here:** [Getting started](getting-started.md) — learn the basics of kdaisyui components and how they work with kotlinx.html

2. **Next:** [Build a dashboard with Ktor](build-a-dashboard.md) — apply what you learned to build a real application with server-side rendering and htmx

## What you'll know after completing these tutorials

- How to add kdaisyui to a Kotlin project
- How to render DaisyUI components with type-safe Kotlin DSL
- How to use variants, sizes, and modifiers
- How to build server-rendered HTML with Ktor
- How to add progressive loading with htmx
- How to structure a real-world application
